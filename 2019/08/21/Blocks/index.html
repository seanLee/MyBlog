Blocks

发表于 2019-08-21 | 更新于 2019-08-27

简介

根据苹果公司的官方文档介绍,Blocks对象具备C语言的语法和runtime的特性.与标准C语言库的函数指针类似,不同点是Blocks同时也会绑定除逻辑代码之外的一些变量.
Blocks不仅可以用于接口的参数传递,属性储存,同时也可以在多线程中使用.当Block作为回调时,由于其同时包含代码执行和持有数据的特性,使用起来会很便捷.
